Usage Guidelines Best practices

✓ Do's

  • Use approved color combinations only
  • Maintain minimum clear space around logo
  • Use high-resolution assets for print (300 DPI+)
  • Apply logo on high-contrast backgrounds
  • Use provided font weights consistently

✗ Don'ts

  • Stretch, distort, or rotate the logo
  • Change the logo colors arbitrarily
  • Place logo on busy or low-contrast backgrounds
  • Add effects like shadows, glows, or outlines
  • Recreate or modify the logo artwork

Minimum Sizes

  • Digital: 100px minimum width
  • Print: 30mm minimum width
  • Favicon: 32×32px
  • Social avatar: 400×400px recommended

File Formats

  • SVG — Web, scalable, editing
  • PNG — Digital with transparency
  • PDF — Print production
  • EPS — Vector editing software
