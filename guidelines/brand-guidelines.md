# CLI Brand Guidelines

## Overview
This document outlines the brand guidelines for CLI, ensuring consistent visual identity across all platforms and materials.

---

## Logo

### Primary Logo
The CLI logo consists of a cloud icon paired with the wordmark "CLI". The cloud represents cloud infrastructure, security, and reliability.

### Logo Variations
- **Silver on Dark** — Primary usage for dark backgrounds
- **Orange on Dark** — For emphasis and call-to-action contexts
- **Dark on Light** — For light/white backgrounds
- **White on Orange** — For branded accent areas

### Clear Space
Maintain minimum clear space equal to the height of the "C" in CLI around all sides of the logo.

### Minimum Size
- Digital: 80px width
- Print: 25mm width

## Typography

### Primary Typeface: Inter Tight
Modern, clean, and highly legible. Used for headings and body text.

| Weight  | Size  | Usage                    |
|---------|-------|--------------------------|
| Bold    | 64pt  | Hero headlines           |
| Medium  | 48pt  | Section headers          |
| Regular | 40pt  | Subheadings              |
| Light   | 36pt  | Body text, descriptions  |

### Secondary Typeface: Helvetica
Classic and universal. Used for technical documentation and fallback.

| Weight       | Size  | Usage                    |
|-------------|-------|--------------------------|
| Bold        | 64pt  | Technical headers        |
| Regular     | 48pt  | Documentation            |
| Bold Oblique| 40pt  | Emphasis                 |
| Oblique     | 36pt  | Captions, notes          |

## Usage Guidelines

### Do's ✓
- Use approved color combinations
- Maintain minimum clear space around logo
- Use high-resolution assets for print
- Apply logo on contrasting backgrounds
- Use provided font weights consistently

### Don'ts ✗
- Stretch or distort the logo
- Change the logo colors arbitrarily
- Place logo on busy backgrounds
- Add effects like shadows or gradients to the logo
- Rotate or flip the logo
